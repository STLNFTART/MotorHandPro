## Authentication Patterns

### API Key (Bearer Token)
```python
def _get_headers(self) -> Dict[str, str]:
    headers = super()._get_headers()
    headers["Authorization"] = f"Bearer {self.credentials['api_key']}"
    return headers
```

### Basic Auth
```python
def _get_headers(self) -> Dict[str, str]:
    import base64
    headers = super()._get_headers()
    creds = f"{self.credentials['username']}:{self.credentials['password']}"
    encoded = base64.b64encode(creds.encode()).decode()
    headers["Authorization"] = f"Basic {encoded}"
    return headers
```

## Webhook Support

All APIs with webhook capabilities include signature verification:

```python
from lam.integrations.api_framework import WebhookReceiver

class SlackWebhookReceiver(WebhookReceiver):
    @property
    def service_name(self) -> str:
        return "slack"

    async def verify_signature(self, payload: bytes, signature: str) -> bool:
        # Verify HMAC signature
        expected = hmac.new(
            self.credentials['signing_secret'].encode(),
            payload,
            hashlib.sha256
        ).hexdigest()
        return hmac.compare_digest(f"v0={expected}", signature)

    async def process_event(self, event_type: str, data: Dict[str, Any]) -> Dict[str, Any]:
        # Handle webhook event
        return {"status": "processed"}
```

## Rate Limiting

Built-in rate limiting for all APIs:

```python
from lam.integrations.rate_limiter import RateLimiter

rate_limiter = RateLimiter(max_requests=100, window=60)  # 100 req/min

async def execute(self, operation: str, **kwargs):
    await rate_limiter.acquire()
    return await super().execute(operation, **kwargs)
```

## Security Best Practices

1. **Never commit credentials** - Use environment variables
2. **Rotate API keys** regularly
3. **Verify webhook signatures** - Always validate incoming webhooks
4. **Use HTTPS only** - Never send credentials over HTTP
5. **Implement rate limiting** - Prevent API abuse
6. **Log sensitively** - Mask credentials in logs
